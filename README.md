## Run Locally

Clone the project

```bash
  git clone https://github.com/pmndrs/market
```

Go to the project directory

```bash
  cd market
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

Some needs to know:

- Login does not work locally
- No environment keys are needed unless you need to do things that change the database
